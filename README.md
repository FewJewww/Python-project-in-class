# Python-project-in-class
My final project in COMP9001

My proposed project is a Personal Pocket Money Management Program called Where is My Money, designed to help users track their spending on hobbies, manage little savings goals, better understand their financial habits and organize their properties.


  1.Spending Analysis: The program will calculate and display the proportion of spending across different categories to help users identify their main expenses.
  
  2.Wish List: Users can set a wish list (the products they really want and their budget) or specific financial goals, and the system will track their progress and estimate how long it will take to achieve the goal, and when they achieve the goal, the system will also compare the budget and actual spendings.
  
  3.Smart Suggestions: Based on spending patterns, the program will provide simple recommendations, such as suggesting areas where users could reduce expenses.
  
  4.Data Persistence: User data (income, expenses, and goals) will be stored and loaded from files, allowing long-term tracking.
  
  5.Property organization: The products the user buyed is also stored and can be organized. And if the user sell the products they owned, the system will compare the money from buying and selling.
